# IPL Toss Analysis

This project analyzes the impact of toss decisions on IPL match outcomes.

## Questions explored

- Does winning the toss improve chances of winning?
- Is it better to bat or bowl after winning the toss?
- What first innings scores are associated with wins and losses?

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Dataset

IPL 2023 Match Dataset

## Key Findings

- Toss winners did not always win the match.
- Teams choosing to bowl showed different success rates than teams choosing to bat.
- First innings score alone was not a guarantee of victory.
